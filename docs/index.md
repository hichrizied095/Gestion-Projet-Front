## Getting Started

1.  **Install Dependencies:**
    ```bash
    npm install
    ```

2.  **Run Development Server:**
    ```bash
    npm start
    ```
    Navigate to `http://localhost:4200/`.

3.  **Build:**
    ```bash
    npm run build
    ```

4.  **Test:**
    ```bash
    npm test
    ```
